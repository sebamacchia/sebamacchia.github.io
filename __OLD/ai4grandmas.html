For several decades the ai was based on "expert systems", that is, super structured computer programs with rules; if ... do ...

With the rise of Machine Learning in the 90s, there is a paradigm shift from which the computer itself learns with numerous examples (a lot of data) what those rules are necessary to solve a particular task.

How does the computer learn by itself?

To develop a machine learning system we need: data, an algorithm that processes that data, and some feedback signal that indicates to the system that it is doing the job well.

For example, if we want to develop a system that detects people's faces in images, firstly, we need a lot of photos where it is highlighted (labeled) where the people's faces are.
Some specific algorithm of image processing is going to be in charge of looking for the faces of the people in the photos.
At the beginning it is likely to do very badly but with a correct feedback signal that rewards hits and penalizes mistakes, the system will be able to correct itself after many repetitions.

Deep learning is an artificial intelligence function that imitates the workings of the human brain in processing data and creating patterns for use in decision making. Deep learning is a subset of machine learning in artificial intelligence (AI) that has networks capable of learning unsupervised from data that is unstructured or unlabeled. Also known as deep neural learning or deep neural network.
